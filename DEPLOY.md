# Deploy no Netlify (mastigado)

## Opção 1 — Drag & drop (mais fácil)
1) No Netlify: **Add new site → Deploy manually**
2) Arraste **a pasta inteira** deste projeto (onde está o `index.html`).
3) Pronto.

## Opção 2 — Definir Publish directory
Se você subir via Git:
- Publish directory: `.`
- (não precisa build command)
